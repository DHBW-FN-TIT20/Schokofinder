# Schokofinder
A small website centered around a chocolate quiz 

# Usage
The dependencies can be installed with npm
```
npm install
```

The server can be started with:
```
node server.js
```

## Required dependencies:
 - node
 - fastify
 - fastify-formbody
 - fastify-static
 - handlebars
 - point-of-view
